EZVD
==========

![EZVD Screencast](https://raw.githubusercontent.com/apollo-ng/ezvd/master/screencast.gif)

## Installation

### Dependencies

  * Python 3.x for full UTF support
  
### Clone repo

    $ git clone https://github.com/apollo-ng/ezvd.git
    $ cd ezvd
    
## Run

    $ ./ezvd.py
